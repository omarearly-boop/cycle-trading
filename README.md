# Cycles Trading Scanner

## Files
- `cycles_trading_scanner.py` — Main scanner (20 Factors)
- `position_manager.py` — Trade management module
- `generate_discord_report.py` — Discord Q&A learning report generator
- `discord_monitor.py` — Discord message extractor
- `preprocess_discord.py` — Message preprocessor
- `watchlists.json` — Watchlist data

## Usage
```
python cycles_trading_scanner.py
python generate_discord_report.py
```
